## Key Decisions

| Decision | Rationale |
|----------|-----------|
| Default AWS region us-east-2 | Bedrock latency-optimized |
| Lazy singleton pattern for AWS clients | Matches existing twilio.ts style |
| Lookup table for mulaw decode | Faster than algorithm |
| 300ms debounce for utterance_end | Responsive yet stable |
| Generative voice engine with Ruth voice | Best conversational quality |
| 100ms audio chunks for TTS streaming | Low-latency delivery |
| Claude 3.5 Sonnet for conversation, Haiku for summaries | Quality vs cost balance |
| Sentence boundary detection for streaming TTS | Natural speech flow |
